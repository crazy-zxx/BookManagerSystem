# 图书管理系统

* 项目描述：
> 使用MVC设计模式简易实现的图书管理系统，包含了图书、读者、管理员三个模块，实现了图书的增删改查，读者的注册、登录、查看借阅记录、注销，管理员的登录、图书管理、读者管理、注销等功能。 

* 开发环境： 
> IDEA + JDK1.8 + MySQL + Tomcat9.0.6 + Struts2 + Spring4 + Hibernate5 + jQuery 

* 项目收获： 
> ①对面向对象的设计思想有了更进一步的理解； <br>
> ②熟悉了JSP标签、Struts标签的使用；  <br>
> ③掌握了SSH框架结合的配置和使用。 <br>
